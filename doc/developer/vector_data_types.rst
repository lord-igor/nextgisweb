Vector layer data types
===============================

.. _ngwdev_field_types:

The following vector layer field types are supported:

* INTEGER - number
* BIGINT - long number
* REAL - floating point number
* STRING - string
* DATE - date
* TIME - time
* DATETIME - timestamp

The following vector layer geometry types are supported:

* POINT
* LINESTRING
* POLYGON
* MULTIPOINT
* MULTILINESTRING
* MULTIPOLYGON
